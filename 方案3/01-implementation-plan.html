6. 部署架构设计

局域网部署架构

设计师工作站 ──→ Nginx反向代理 ──→ 前端应用/后端API │ ├──→ 外部服务 (Casdoor/Gitea/Mattermost) │ └──→ 数据存储 (PostgreSQL/Redis/MinIO)

systemd服务配置示例

# /etc/systemd/system/rdp-api.service [Unit] Description=RDP Platform API Service After=network.target postgresql.service redis.service [Service] Type=simple User=rdp-user Group=rdp-user WorkingDirectory=/opt/rdp ExecStart=/opt/rdp/bin/rdp-api Restart=always RestartSec=5 MemoryMax=2G # 环境变量 Environment="DB_HOST=127.0.0.1" Environment="DB_PORT=5432" Environment="DB_NAME=rdp_db" Environment="REDIS_HOST=127.0.0.1" Environment="PORT=8080" [Install] WantedBy=multi-user.target
